== Description ==

Ahime Image Printer is a tool dedicated to photographers. It will allow your customers to resize your photos according to certain ratio.

It has several features such as:

* Resize image
* Crop image
* Change Image orientation (portrait or landscape)
* Add unlimited image
* Change image order quantity
* Add resizes image to cart

Tested on wordpress version higher than 5, it is compatible with the different version of woocommerce.
The most stable version of the plugin is the first version 1.0.0

== Installation ==

1. Upload `ahime-image-printer` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Ahime > settings and define your personalization page
4. Create a product
5. Just below the price section you will see "Printing Image config", select yes
